## Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/matheushenriquecsb/booking-frontend
```

### 2. Acesse o diretório

```bash
cd booking-frontend
``` 

### 3. Instalando dependências

```bash
npm install
```

### 4. Rode a aplicação

```bash
npm run dev
```

### 5. Acesse a aplicação

```bash
http://localhost:5173
```
